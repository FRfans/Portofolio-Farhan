# Farhan Premium Portfolio

Website portfolio statis berbasis HTML, CSS, dan JavaScript vanilla.

## Cara menjalankan
1. Ekstrak ZIP.
2. Buka `index.html` di browser.
3. Edit data project, skill, sertifikasi, pengalaman, process, dan testimonial di `script.js`.
4. Edit tampilan di `style.css`.

## Deploy
Bisa langsung di-deploy ke GitHub Pages, Netlify, atau Vercel sebagai static site.

## Catatan
Link CV mengarah ke `assets/Farhan_Rahmansyah_CV.pdf`. Buat folder `assets`, lalu masukkan file CV dengan nama tersebut agar tombol Download CV aktif.
